# WRTN RP Memory Compressor

WRTN RP 장기기억 압축 보조 스크립트입니다.

## 주요 기능

* 장기기억 자동 펼치기
* 장기기억 일괄 수집
* 장기기억 건강도 표시
* 압축 프롬프트 자동 첨부
* 클립보드 복사

## 사용 방법

1. 장기기억 탭 진입
2. 압축할 기억까지 스크롤하여 로딩
3. 🧠 압축 복사 버튼 클릭
4. 외부 LLM(ChatGPT, Gemini 등)에 붙여넣기
5. 압축 결과 수신
6. 압축 결과를 바탕으로 직접 장기기억 수정

## 장기기억 건강도

🟢 0~19개 : 안전

🟡 20~29개 : 주의

🟠 30~49개 : 압축 권장

🔴 50개 이상 : 압축 추천

## 주의사항

* 이 스크립트는 장기기억을 읽고 복사만 수행합니다.
* 장기기억 수정, 삭제, 추가 기능은 포함되어 있지 않습니다.
* 압축 적용은 사용자가 직접 수행해야 합니다.
* 스크롤되지 않은 기억은 수집되지 않을 수 있습니다.
* 사용 전 압축할 기억까지 충분히 스크롤해 주세요.

## 설치 방법

1. Tampermonkey 설치
2. 새 스크립트 생성
3. WRTN_RP_Memory_Compressor.user.js 내용 붙여넣기
4. 저장
5. WRTN RP 페이지 새로고침
